Q5:What is the Box Model??


Box Model contains the css properties like Margin,Padding,Border
Margin is used to move element to any direction .. for example if we use margin-left:10px then it will takes that particular element to 100px right from current position
Padding is used to Take space from all sides(internally)
Border is used to give border to an element for example.. border="20px"




Q6: what is float left and float right


Float Left is a css property used to float the particular element to left side of browser by using float:left
Float Right is a css property used to float the particular element to right side of browser by using float:right
